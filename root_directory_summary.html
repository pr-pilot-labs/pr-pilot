README.md

PR Pilot is a developer tool/platform that assists developers in their daily work and integrates with the dev tools they use daily. It supports Django/Python/Docker/Kubernetes and provides various ways to interact, including CLI, Python SDK, REST API, and Smart Workflows. The project is open source and available under the GPL-3 License.

CONTRIBUTING.md

This file provides guidelines for contributing to the PR Pilot project. It includes instructions on reporting bugs, discussing the current state of the code, submitting fixes, proposing new features, and becoming a maintainer. The project uses GitHub Flow, and all code changes happen through pull requests.

Makefile

The Makefile contains various build and deployment commands for the project. It includes targets for building Docker images, pushing them to a registry, deploying the application using Helm, and managing Kubernetes secrets. It also has commands for running unit tests and generating PR descriptions.

Dockerfile

The Dockerfile sets up the environment for running the PR Pilot application. It uses a Python 3.11-slim base image and installs necessary dependencies, including uWSGI for serving the application. The Dockerfile also includes steps for installing system dependencies, setting environment variables, and copying the project files.

version.txt

This file contains the current version of the PR Pilot project, which is 1.9.10.
