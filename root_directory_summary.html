Project Overview

This project, PR Pilot, is a developer tool/platform that assists developers in their daily work and integrates with the dev tools they use daily. It is built using Django, Python, Docker, and Kubernetes.

Files in the Root Directory

README.md

The README file provides an overview of the PR Pilot project, including installation instructions, usage examples, and links to documentation and other resources. It also includes badges for build status, Python version, Django version, license, and project version.

CONTRIBUTING.md

This file contains guidelines for contributing to the PR Pilot project. It outlines the process for reporting bugs, discussing code, submitting fixes, proposing new features, and becoming a maintainer. It also provides instructions for using GitHub Flow and writing detailed bug reports.

Makefile

The Makefile contains various commands for building, deploying, and managing the PR Pilot project. It includes targets for building static files, Docker images, pushing images to a registry, deploying the application using Helm, creating Kubernetes secrets, running migrations, and generating PR descriptions and daily reports.

requirements.txt

This file lists the Python dependencies required for the PR Pilot project. It includes packages such as Django, PyGithub, langchain, openai, GitPython, beautifulsoup4, kubernetes, jinja2, psycopg2-binary, djangorestframework, and more.

version.txt

The version.txt file contains the current version number of the PR Pilot project, which is 1.9.10.
